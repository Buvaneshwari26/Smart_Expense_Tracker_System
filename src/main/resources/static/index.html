Technical Architecture

A clean, production-ready full stack ecosystem built with modern guidelines.

Backend Architecture

Engineered with **Spring Boot 3.2.5** targeting Java 21 bytecode compatibility. It employs:

  • **Spring Security** & **JWT**: Stateless token authorization structure.
  • **Spring Data JPA**: Soft-deletes (`@SQLDelete`), auditing, sorting, pagination.
  • **Swagger UI**: API exploration console via OpenAPI 3.
  • **Docker & docker-compose**: MySQL 8 and application orchestration.
Frontend Stack

Avoids complex node modules compile overhead using high performance vanilla scripting:

  • **Bootstrap 5**: Structural grid layers and responsive elements.
  • **Chart.js**: Client-side rendering of analytical models.
  • **Vanilla JS (ES6)**: Reusable API interceptors and JWT token lifecycle managers.
  • **CSS3 Glassmorphism**: Tailored light/dark colors and shimmer skeleton blocks.
